## ✨ 프로젝트 소개

### 🌟 제목

-   리그 오브 레전드 챔피언 및 아이템 도감 커뮤니티 "League Of Legend"

### 🚀 목적

-   오픈 소스를 활용한 개발 학습

### 🤝 팀 프로젝트

-   팀원 2명

### 📆 제작 기간

-   2023.12.11 ~ 2024.01.15

### 🔎 주요 기능

-   회원가입 및 로그인
-   Firebase Authentication을 통한 소셜 로그인
-   챔피언 도감

    > 챔피언 검색

    > 챔피언 상세 정보

    > 챔피언 댓글 등록

-   아이템 도감

    > 장바구니 추가

    > 장바구니 확인

-   마이페이지

    > 프로필 아이콘 변경

    > 닉네임 변경

    > 배경화면 변경
